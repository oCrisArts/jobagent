# 📊 Business Model Canvas - Sync.IA Global

## 🎯 Objetivo
Definir modelo de negócio GLOBAL: O que oferecemos, para quem (profissionais corporativos globais) e como monetizamos em USD.

## 🏗️ Estrutura BMC

### **Segmentos de Clientes (GLOBAL)**

**Primário** (60%):
- 👨‍💻 **Engineers/Developers** (USA, EU, latam)
- 🎨 **UX/UI Designers** (Design-forward companies)
- 📊 **Product Managers** (Tech, E-commerce, SaaS)

**Secundário** (40%):
- 👨‍📈 **Data Analysts** (SQL/Analytics roles)
- 💼 **Executives/CTOs** (Hiring better talent)
- 👩‍🎓 **Career Coaches/Recruiters** (B2B2C)

**Geografia**: USA (40%), EU (35%), LATAM (15%), APAC (10%)

### **Proposta de Valor**

- ✅ Adapte CV com IA (Gemini + Claude) - Qualquer profissão
- ✅ SSI Score gamificado (engajamento)
- ✅ Hub de Headhunters/Recruiters (networking global)
- ✅ Job Fit Analysis (múltiplos mercados)
- ✅ Suporte multi-idioma (EN + PT)
- ✅ Preço acessível em USD ($14.90/mês)

### **Canais**

- **Aquisição**: LinkedIn Ads, Product Hunt, Hacker News, Reddit
- **Retenção**: Email, daily missions, SSI Score, push notifications
- **Monetização**: Paywall (Stripe, suporta USD + múltiplas moedas)

### **Relacionamento com Clientes**

- **Free**: Self-service + email support (EN + PT)
- **Pro**: Priority support + relatórios em PDF
- **Enterprise**: Account manager + API customizado

### **Fontes de Receita**

- **Tier Free**: $0 (aquisição)
- **Tier Pro**: **$14.90/mês** (5-8% conversão target)
- **Tier Enterprise**: $499-999/mês (coaches/enterprises)

**Exemplo MRR**: 
- 1.000 signups → 5% Pro = 50 Pro users
- 50 × $14.90 = **$745/mês MRR**

### **Recursos Principais**

- Banco de dados (Supabase - global)
- IA APIs (Gemini + Claude - USA-based)
- Stripe (pagamento global, USD)
- Frontend (Next.js + next-intl)
- CDN global (Vercel)

### **Atividades-Chave**

- Desenvolvimento product (agile)
- Marketing de aquisição (LinkedIn Ads global)
- Suporte ao cliente (24/7, multi-idioma)
- Análise de dados (HEART metrics)

### **Parcerias**

- Supabase (backend)
- Stripe (pagamento)
- Google (Gemini)
- Anthropic (Claude)
- Vercel (hosting global)

## 🎯 Breakeven (Ponto de Equilíbrio)

### **Matemática Realista:**
```
Custos Mensais: $168
Preço por usuário: $14.90
Unidades para breakeven: $168 / $14.90 = 11.27 usuários

BREAKEVEN: ≈ 12 usuários Pro/mês
```

### **Comparação com Expectativa:**
```
Breakeven alcançado com apenas:
  ✅ 12 usuários Pro (1.2% conversão de 1.000 signups)
  ✅ Em menos de 2 meses (startup realista)
  ✅ Com margem de 88% acima do breakeven

Versus:
  ❌ Modelo SaaS tradicional: 100-500 usuários
  ❌ Plaforma escalável: 2-3 anos para profitabilidade
  ❌ Margens degradam com crescimento
```
