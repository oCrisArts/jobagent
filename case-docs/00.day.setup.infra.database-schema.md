# 🗄️ Database Schema - PostgreSQL Supabase

## 🎯 Objetivo
Estruturar o banco de dados completo com tabelas, enums, triggers, RLS e views.

## 📊 Tabelas Criadas

### Core Tables
- **profiles** - Usuários com flags is_pro, subscription_status
- **resumes** - PDFs e versões de currículo
- **job_applications** - Histórico de candidaturas
- **headhunters** - CRM de contatos
- **gamification_missions** - Missões diárias
- **subscriptions_audit** - Auditoria de pagamentos
- **pricing_plans** - Planos Free e Pro

## 🔐 Segurança
- ✅ RLS ativado em todas as tabelas
- ✅ Políticas: usuários veem apenas seus dados
- ✅ Triggers: updated_at automático
- ✅ Constraints: validações de dados
- ✅ Indexes: otimização de queries

## 📈 Views Úteis
- `user_dashboard_summary` - Resumo para HUD
- `recent_applications` - Apps recentes
