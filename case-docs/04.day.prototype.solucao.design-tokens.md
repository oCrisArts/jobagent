# 🎨 Design System & Tokens - Figma

## 🎯 Objetivo
Definir estética Tech-Noir/Cyberpunk e criar design tokens para Figma

## 🌈 Paleta de Cores

### Dark Theme (Primary)
```
Background Primary:  #0F0F0F (Almost black)
Background Secondary: #1A1A1A (Dark gray)
Surface Tertiary:    #2A2A2A (Medium gray)

Primary Brand:       #00D9FF (Cyan/Neon)
Secondary:           #FF006E (Neon Pink)
Tertiary:            #8000FF (Neon Purple)

Success:             #00FF41 (Neon Green)
Warning:             #FFB300 (Neon Orange)
Danger:              #FF0055 (Neon Red)

Text Primary:        #FFFFFF (White)
Text Secondary:      #B0B0B0 (Light Gray)
Text Tertiary:       #707070 (Medium Gray)
```

### Light Theme (Secondary)
```
Background:          #FAFAFA
Surface:             #FFFFFF
Primary:             #0099CC (Blue)

(Accessible but not prioritized in MVP)
```

## ✍️ Typography

```
Headings:   "Inter Bold" / 600 weight
- H1: 32px, line-height: 1.2
- H2: 24px, line-height: 1.3
- H3: 20px, line-height: 1.4

Body:       "Inter Regular" / 400 weight
- Large:  16px, line-height: 1.5
- Normal: 14px, line-height: 1.5
- Small:  12px, line-height: 1.4

Mono:       "JetBrains Mono"
- For code snippets
- For numbers (SSI Score)
```

## 📦 Component System

### Buttons
```
Primary (CTA):
- Background: #00D9FF (Cyan)
- Text: #0F0F0F (Black)
- Padding: 12px 24px
- Border-radius: 8px
- Hover: Brightness +10%

Secondary:
- Background: transparent
- Border: 1px #00D9FF
- Text: #00D9FF

Disabled:
- Background: #2A2A2A
- Text: #707070
```

### Cards
```
Background: #1A1A1A
Border: 1px solid #2A2A2A
Border-radius: 12px
Padding: 16px
Box-shadow: 0 2px 8px rgba(0,217,255,0.1)

Hover: 
- Border: 1px solid #00D9FF
- Box-shadow: 0 4px 16px rgba(0,217,255,0.2)
```

### Input Fields
```
Background: #2A2A2A
Border: 1px solid #404040
Border-radius: 8px
Padding: 10px 12px
Text: #FFFFFF

Focus:
- Border: 2px solid #00D9FF
- Box-shadow: 0 0 8px rgba(0,217,255,0.3)
```
