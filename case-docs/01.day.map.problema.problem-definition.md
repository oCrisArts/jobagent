## 📌 Problem Statement (Global)

### **O Problema**

Profissionais corporativos (Engenheiros, PMs, Designers, Analistas, Executives) em TODO O MUNDO perdem **oportunidades de carreira** porque:

1. **CV Genérico**
   - Mesmo CV para todas as vagas (especialmente em mercado global)
   - ATS systems não capturam skills reais
   - Palavras-chave não matcham com job descriptions
   - Resultado: Rejeitados antes de serem vistos por recrutadores

2. **Sem Análise de Fit**
   - Candidatam para vagas inadequadas (waste de tempo)
   - Não sabem se têm 80% match ou 30% match
   - Trabalho de casa (research + customização) = 2-3h por vaga
   - Globalmente: Múltiplos mercados, múltiplas moedas, múltiplas culturas

3. **Networking Desorganizado**
   - Recruiter network é chaotic (LinkedIn + emails dispersos)
   - Sem follow-up sistemático
   - Sem visibilidade de relacionamentos
   - Global recruitersmundo afora = impossível gerenciar manualmente

4. **Falta de Gamificação / Motivação**
   - Job search é depressivo (high rejection rate)
   - Sem indicadores de progresso
   - Sem feedback rápido
   - Sem comunidade ou suporte

### **Impacto por Persona**

**Engineers** (Alex - USA):
- Perde $10-20K em salary negotiation (bad positioning)
- Gasta 50+ horas/mês em applications ineficazes
- Não consegue destacar impacto técnico

**Product Managers** (Sofia - Germany):
- Vagas PM desejáveis (FAANG, startups) ficam invisíveis
- CV não comunica "Impact at Scale"
- Sem dados para validar fit

**Designers** (Marcus - Canada):
- Portfolio doesn't translate to CV
- Design background confuses technical recruiters
- Hard to position for global remote roles

**Recruiters** (Elena - UK):
- Manual process = não escala
- Candidate assessment = adivinhar
- Sem ROI claro para recruiting firm

## 💡 Solução Proposta: Sync.IA

**Um gestor de carreira globalizado + gamificado que**:

1. ✅ **Adapte CV com IA**
   - Gemini (free): Extração + adaptação básica
   - Claude (pro): Adaptação premium com industry insights
   - Suporta EN + PT (extensível)

2. ✅ **Analise Job Fit em Tempo Real**
   - Calcula % match (80% → high chance)
   - Mostra gaps de skill
   - Prioriza vagas por fit

3. ✅ **Hub de Recruitersgamificado**
   - Organize networking
   - Follow-up automático
   - Trust score visual
   - Global recruiter database

4. ✅ **Gamifique a Jornada**
   - SSI Score (0-100)
   - Daily missions
   - Levels + XP
   - Motivação constante

5. ✅ **Global-First**
   - Preço em USD (standard internacional)
   - Multi-idioma (EN + PT)
   - Suporta múltiplos mercados
   - Recruitersglobal
