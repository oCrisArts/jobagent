# 🎬 Protótipo de Alta Fidelidade - Next.js Build

## 🎯 Objetivo
Codificar o suficiente para validar a estética Tech-Noir/Cyberpunk

## 🏗️ Estrutura de Componentes

### Pages
- `pages/landing.tsx` - Hero + Pricing preview
- `pages/pricing.tsx` - Pricing page completa
- `pages/auth/signup.tsx` - Signup modal
- `pages/iniciar.tsx` - HUD principal
- `pages/buscar.tsx` - Job search
- `pages/evoluir.tsx` - Resume upload
- `pages/conectar.tsx` - Headhunter hub
- `pages/checkout.tsx` - Stripe integration

### Components
- `Button` - Primary, Secondary, Disabled
- `Card` - Pricing, Job, Headhunter
- `SSIScore` - Circular progress
- `JobFitBadge` - % visual
- `FormInput` - Email, password
- `Modal` - Generic modal wrapper

### Hooks
- `useAuth()` - Auth context
- `useJobs()` - Job API
- `useResumeUpload()` - Upload logic
- `useSSIScore()` - Score calculation
