# 📊 HEART Metrics & Analytics - Dashboard

## 🎯 Objetivo
Definir métricas para medir sucesso do Sync.IA

## 📈 Dashboard HEART

### Happiness (Satisfação do usuário)
```
NPS (Net Promoter Score)
├─ Pergunta: "0-10, recomendaria?"
├─ Calculation: (Promoters% - Detractors%)
└─ Target: 7+

Satisfaction
├─ Pergunta: "Resolveu seu problema?"
├─ Scale: 1-5 (muito insatisfeito a muito satisfeito)
└─ Target: 4+

Feature Satisfaction
├─ "CV Adaptation foi útil?"
├─ "SSI Score te motivou?"
└─ Target: 4+
```

### Engagement (Uso frequente)
```
DAU (Daily Active Users)
├─ Usuários ativos por dia
├─ Calculation: unique users on day
└─ Target: 20% of installs

MAU (Monthly Active Users)
├─ Usuários ativos por mês
└─ Target: 60%+ retention

Feature Engagement
├─ Job Search: 70%+ de users
├─ Adaptation: 50%+
├─ Headhunter Hub: 30%+ (Pro)
└─ SSI Score daily check: 80%+
```

### Adoption (Novos usuários)
```
Signup Rate
├─ Signups por dia
└─ Target: 100+ mês 1

Onboarding Completion
├─ Upload PDF → First Adaptation
└─ Target: >70%

First Adaptation Rate
├─ % que fazem primeira adaptação
└─ Target: >60% of signups
```

### Retention (Volta usuários)
```
D1 Retention: 70%
D7 Retention: 40%
D30 Retention: 20%

Churn Rate
├─ % que cancela subscription
├─ Target: <5% (Pro)
└─ Calculation: churn = (lost customers / start customers)
```

### Task Success (Conversão)
```
Signup to First Adaptation: 70%+
First Adaptation to Upgrade: 5%+
Free User to Pro: 2-5%+
Upgrade Complete: 90%+
```

## 📊 Dashboards Recomendados

### Google Analytics
- Real-time: Usuarios agora
- Audience: Demographics
- Acquisition: Source de traffic
- Behavior: Pages mais visitadas
- Conversions: Funnel upgrade

### Hotjar
- Heatmaps: Click, move, scroll
- Session Recordings: Vídeos de usuários
- Feedback Polls: "Qual a maior dificuldade?"
- Funnels: Onde saem users

### Clarity
- Session Replay: Vídeos Microsoft
- Funnel Analysis: Drop-off points
- User Journeys: Paths populares
